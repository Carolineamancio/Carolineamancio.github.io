## 🌟 Sobre o Projeto

Este é um portfólio pessoal bonito e responsivo criado para showcasear minha paixão por desenvolver jogos. O site apresenta uma interface moderna com tema roxo e rosa, perfeito para uma desenvolvedora de jogos que adora criar experiências memoráveis.

O design é **premium**, **responsivo** e otimizado para todos os dispositivos, desde desktops até smartphones.


- **CSS Grid** e **Flexbox** para layouts responsivos
- **Variáveis CSS** (CSS Custom Properties) para tema consistente
- **Gradientes** lineares e radiais
- **Backdrop Filter** para efeito glassmorphism

Acesse diretamente em: [https://carolineamancio.github.io/](https://carolineamancio.github.io/)

## 💡 Dicas de Customização

### Mudar Cores
Edite as variáveis CSS em `index.html`:
```css
:root {
  --accent: #ff5fa2;        /* Rosa primário */
  --accent-2: #9b5cff;      /* Roxo primário */
  --text: #fff7ff;          /* Texto claro */
  /* ... outras variáveis */
}
```

### Adicionar Novos Jogos
1. Crie um arquivo HTML para o novo jogo
2. Adicione um novo `project-card` na seção "Projetos"
3. Atualize os links nos cards

### Personalizar Conteúdo
Edite o texto nas seções relevantes:
- **Hero** - Apresentação pessoal
- **Sobre** - Informações profissionais
- **Projetos** - Descrição dos jogos
- **Contato** - Links e redes sociais
